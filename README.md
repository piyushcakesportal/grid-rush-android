# Grid Rush Android

Native Android test game.

## Rules
Tap a connected group of 3 or more matching tiles. Bigger groups and consecutive successful moves score more points. You have 45 seconds.

## Build APK using GitHub Actions
1. Create a GitHub repository.
2. Upload all files from this project.
3. Open **Actions → Build Grid Rush APK → Run workflow**.
4. Open the completed run.
5. Download the `grid-rush-apk` artifact.
6. Extract it and install `app-debug.apk` on Android.

The APK is a debug test build and is not suitable for Google Play publishing.
